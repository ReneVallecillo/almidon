Configurando

Aunque es posible directamente acceder a classes/config.php y a classes/tables.class.php y hacer las modificaciones correspondientes. Es posible tambien acceder directamente a http://example.com/setup y realizar desde ahi varias de las actividades.

Creando un objeto

Usando new tableTable(); a como es definido en tables.class.php Ejemplo, crea objeto par tabla noticia: 
  $data = new noticiaTable();

Leyendo datos

readData
Lee todos los datos de una tabla. Ejemplo, leyendo todos los datos de enlace: 
  $data = new enlaceTable();
  $rows = $object->readData()
  print_r($rows);
readDataFilter
Lee datos de una tabla aplicando un filtro Ejemplo, lee todos los eventos del dia 2007-10-13: 
  $data = new agendaTable();
  $rows = $object->readDataFilter("fecha='2007-10-13'");
  print_r($rows);
readRecord
Lee un registro de la tabla Ejemplo, lee registro pagina con Id = 'quienes_somos': 
  $data = new paginaTable();
  $row = $object->readRecord('quienes_somos');
  print_r($row);
