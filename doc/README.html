Sobre almidon

Almidón es un framework de desarrollo rápido y sistema de manejo de contenido, publicado bajo la licencia open source GPL v3. Ver LICENSE Mantiene su wiki en http://almidon.org/

Almidón es tambien una plataforma de hosting, aunque es por ahora la parte menos documentada, la idea es contar con un framework central en un servidor que permita tener varios proyectos funcionando con el mismo framework, sin tener que mantener cada sitio web por separado.

Para entender mejor almidon ver ENTENDIENDO

Requerimientos

Almidon es desarrollado para trabajar en Linux, Apache, Postgresql, usando PHP. Sin embargo es flexible para adaptarse a otras configuracion, siendo la más común el cambio de Postgresql por Mysql. El detalle de los requerimientos son:

  • Apache 1.3+
  • PHP 4.3+ (indispensable)
  • PHP GD para manejo de imagenes
  • Mysql 4+ o PostgreSQL 7+
  • Unix/Linux (para sacar provecho de symlinks)

Almidon ha sido probado en plataformas muy disintas, siendo PHP el único requisito indispensable, ha sido probado por ejemplo en Windows + IIS + SQL Server. Sin embargo lo especificado anteriormente es lo minimo necesario para sacar provecho máximo de por ejemplo symlinks, mod_rewrite, etc.

Quienes somos?

Actualmente almidón es primordialmente desarrollado por Guegue.Com y usado intensivamente en cientos de sitios web, desarrollados y en su mayoría hospedados por Guegue.Com Desde abril 2008, almidon comienza a desarrollarse con apoyo de la comunidad de software libre en Nicaragua. Desde el 2009 recibe tambien colaboracion de un grupo cubano de desarrolladores.
