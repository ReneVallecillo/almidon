Utilizando almidon

Almidon puede ser utilizado en distintos escenarios como herramienta de desarrollo agil. Aqui detallo algunas:

Probarlo
Lo mejor es usar el demo, seguir instrucciones en INSTALL.
En caso de no poder / querar hacerlo se puede usar el demo en linea. http://demo.almidon.org/

Para crear una base de datos y trabajar en ella
Aqui lo mas sencillo es basarse en el demo, o usar site-setup.sh modificar alm_table con las tables y campos requeridos. Luego desde el setup, generar tables.class.php desde alm_tables. Finalmente aplicar cambios a BD desde tables.class.php

Para trabajar en una base de datos ya existente
Igual, basarse en demo o usar site-setup.sh, luego configurar classes/config.php para poder acceder a tu BD. Luego puedes decidir si "almidonizas" (desde el setup) tu BD o no.
